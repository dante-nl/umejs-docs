# umejs documentation

**Available on [ume.dantenl.com](https://ume.dantenl.com)!**

Documentation for umejs, a simple Node project that allows you to serve Markdown pages using express!

The umejs docs have been built using umejs, and if you're interested in using it yourself, feel free to copy some bits of code and just generally use it as an example. If you're interested in umejs, [check out the repo](https://github.com/dantenl/umejs).
